# Codex Workspace Bridge

このワークスペースでは、Codex 用の詳細設定を `agent/.codex/` で共有管理している。

最初に次を読むこと。

1. `agent/AGENTS.md`
2. `agent/.codex/AGENTS.md`

利用可能な role や config は次を参照すること。

- `agent/.codex/config.toml`
- `agent/.codex/agents/*.toml`
- `agent/.agents/skills/*/SKILL.md`

このファイル自体は入口のみを担い、詳細ルールの本体は `agent/.codex/` 側に置く。
